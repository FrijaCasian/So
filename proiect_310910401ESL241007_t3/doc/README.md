#General directory purpose:

##bin/ - executables
##src/ - C source files
##include/ - header files
##data/ - persistent files
##logs/ - execution logs
##reports/ - generated reports
##tmp/ - temporary artifacts
##tests/ - test scenarios
##doc/ - documentation
##tools/ - scripts

#How to execute audit:
```
 bash tools/t1_audit.sh
```
